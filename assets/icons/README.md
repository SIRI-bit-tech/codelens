# CodeLens Icons

Place the following icon files in this directory:

- `app_icon.png` - 512x512 PNG icon for the application
- `app_icon.ico` - Windows icon file
- `app_icon.icns` - macOS icon file

You can generate these icons from a single PNG using online tools or:
- Windows: Use `convert` from ImageMagick or online converters
- macOS: Use `iconutil` or online converters

For now, the application will run without custom icons and use system defaults.
